zagros

a simple operating system from scratch.

prerequisites

  • bear
  • as
  • gcc
  • qemu(or virtualbox or bochs)

optional

diff tools

  • gitk
  • meld

documentation

  • doxygen

static analyzers

  • cppcheck
  • frama-c
  • splint

code coverage

  • kcov

memory analyzer

  • valgrind(memcheck)
  • callgrind
  • cachegrind
  • kcachegrind
  • helgrind
  • massif

utils

  • scc
  • fzf
  • ambr
  • neovim

binary utils

  • addr2line
  • nm
  • objdump
  • readelf
  • size
  • strings

libs

  • man-pages-devel man <method>
  • libsanitizer-devel
  • libsanitizer-devel-32bit

build/run/test

./p
